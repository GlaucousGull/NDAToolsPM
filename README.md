# NDAToolsPM
TCP/UDP 跨平台网络调试助手（Qt Widgets）

## 项目简介
NDAToolsPM 是基于 **Qt Widgets** 开发的桌面端网络调试工具，聚焦 TCP/UDP 协议调试场景，集成服务端、客户端、进制数据转换、数据校验四大模块，是嵌入式、后端网络开发常用的报文收发调试工具。
软件开箱即用，无需额外依赖，支持Windows/Linux跨平台编译运行。

## 功能分区
### 一、四大通讯调试面板
1. **TCP 服务器**：快速监听本地端口，创建TCP服务，管理客户端接入/断开、实时收发十六进制/原始字符串报文；
2. **TCP 客户端**：填写服务端IP+端口，建立/断开TCP连接，自定义发送测试数据、查看返回报文；
3. **UDP 服务器**：绑定本机端口，被动接收全网UDP数据包，支持回复指定源IP报文；
4. **UDP 客户端**：指定远端IP与端口，无连接式发送UDP报文，监听本机回包数据。

### 二、内置数据工具箱（截图对应模块）
1. **数据转换**：多进制互转，支持十进制↔二进制↔十六进制↔ASCII字符串，实时统计输入输出字符数、字节数；
2. **数据校验**：内置常用校验算法（CRC、和校验等），快速对报文做校验值计算，适配串口/网络协议校验开发。

## 技术栈
- 开发框架：Qt 5/Qt6 Widgets
- 编程语言：C++
- 编译环境：MinGW / GCC
- 适用系统：Windows、Linux 桌面系统
